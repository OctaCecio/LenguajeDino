<Trampa_Mortal> = Lava(3)

MAPA Nivel_Roto { 
    Agua(2), 
    <Trampa_Mortal>, 
    Aguila, 
    Pasto, 
    Nave 
}

JUEGO Dino_Noob { 
    Nadar.mientras(Agua), 
    Saltar.simple, 
    Saltar.triple.si_es(Paloma), 
    Correr, 
    Correr 
}

Validar ( Nivel_Roto , Dino_Noob )



El entorno MAPA debe comenzar con al menos un elemento Pasto.

Solo Agua y Pasto aceptan números.

No puede haber más de una lava, paloma o águila contiguas.

No se puede saltar inmediatamente después de salir del agua.

Modificador para salto erróneo (la paloma es doble).

Escape final Despegar != Correr.
